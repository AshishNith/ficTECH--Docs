LED Blink

Components Needed:

  • Arduino board
  • LED
  • 220-ohm resistor
  • Breadboard
  • Jumper wires

Steps:

  1. Connect the LED's anode (longer leg) to a digital pin (e.g., pin 13) through a 220-ohm resistor.
  2. Connect the LED's cathode (shorter leg) to the ground (GND) on the Arduino.
  3. Upload the following code to the Arduino:
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
