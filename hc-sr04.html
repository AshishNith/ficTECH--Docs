HC-SR04 Module

Introduction

Hey there! Are you curious about how to measure distances without touching anything? The HC-SR04 ultrasonic sensor is your answer! This handy little device uses sound waves to measure distance, just like how bats navigate. In this guide, I'll walk you through everything you need to know about the HC-SR04, even if you're completely new to this.

1. What is an Ultrasonic Sensor?

An ultrasonic sensor measures distance by using sound waves. It sends out a sound wave at a frequency higher than humans can hear, waits for the sound to bounce back, and then calculates how far away the object is based on the time it took for the echo to return.

2. Overview of HC-SR04 Ultrasonic Sensor

The HC-SR04 is a popular ultrasonic sensor that’s super easy to use with microcontrollers like Arduino. It's accurate, affordable, and can measure distances from 2 cm to 400 cm. Let’s dive into the details!

Key Features

  • Range: 2 cm to 400 cm
  • Accuracy: ±3 mm
  • Operating Voltage: 5V
  • Easy to interface with Arduino and other microcontrollers

3. Pin Configuration

The HC-SR04 has four pins that are simple to connect. Here’s what they do:

Pins Description

  • VCC: Connects to the 5V power supply.
  • Trig: Trigger pin that sends the ultrasonic pulse.
  • Echo: Receives the reflected pulse.
  • GND: Connects to ground.

4. How It Works

The HC-SR04 works by sending out a short ultrasonic pulse from the trigger pin. This pulse travels through the air, hits an object, and then bounces back to the sensor. The sensor listens for this echo on the echo pin and measures the time it took for the pulse to return. Using this time, it calculates the distance to the object.

Simple Steps

  1. Send a pulse: The sensor sends out a sound wave.
  2. Wait for echo: The sensor listens for the echo.
  3. Measure time: It measures how long it took for the echo to come back.
  4. Calculate distance: Using the speed of sound, it calculates how far away the object is.

5. Wiring and Connections

Let's get you connected! Here’s how to hook up the HC-SR04 to an Arduino.

Components Needed

  • HC-SR04 Ultrasonic Sensor
  • Arduino Board
  • Connecting Wires

Wiring Diagram

  1. Connect VCC to the 5V pin on the Arduino.
  2. Connect GND to the ground pin on the Arduino.
  3. Connect Trig to a digital pin on the Arduino (e.g., pin 9).
  4. Connect Echo to a digital pin on the Arduino (e.g., pin 10).

7. Code Examples for Arduino

Let’s write some code! Below is a simple example to get you started with measuring distance using the HC-SR04.

Basic Example to Measure Distance


    const int trigPin = 9;
    const int echoPin = 10;
    long duration;
    int distance;
    
    void setup() {
        pinMode(trigPin, OUTPUT);
        pinMode(echoPin, INPUT);
        Serial.begin(9600);
    }
    
    void loop() {
        // Clear the trigPin
        digitalWrite(trigPin, LOW);
        delayMicroseconds(2);
    
        // Set the trigPin HIGH for 10 microseconds
        digitalWrite(trigPin, HIGH);
        delayMicroseconds(10);
        digitalWrite(trigPin, LOW);
    
        // Read the echoPin, returns the sound wave travel time in microseconds
        duration = pulseIn(echoPin, HIGH);
    
        // Calculate the distance
        distance = duration * 0.034 / 2;
    
        // Print the distance on the Serial Monitor
        Serial.print("Distance: ");
        Serial.print(distance);
        Serial.println(" cm");
    
        delay(1000); // Wait for a second before the next measurement
    }

8. Troubleshooting

Common Issues

  • No Reading: Make sure the sensor is powered correctly and the pins are connected properly.
  • Inaccurate Measurements: Ensure there are no obstacles in front of the sensor, and it is facing a flat surface.
  • Interference: Ultrasonic sensors can interfere with each other. If you have multiple sensors, try using them one at a time.

Q1: Can the HC-SR04 work in the dark?
A1: Yes, the HC-SR04 works perfectly in the dark since it uses sound waves, not light.

A2: The HC-SR04 can measure distances up to 400 cm.
Q3: Can I use the HC-SR04 underwater?
A3: No, the HC-SR04 is not designed for underwater use.
Q4: How accurate is the HC-SR04?
A4: The HC-SR04 is accurate within ±3 mm.
